# 🌟 Mamidi Chandra Mohan | Professional Portfolio

A high-performance, modern professional portfolio built for **Mamidi Chandra Mohan**, a distinguished Senior Educator with over 30 years of experience in Hindi and Telugu literature.

> "Chukkallo Chandrudu" — *The Moon among the Stars.*

---

## 🚀 Overview

This website is designed to showcase a legendary career in education, spanning from 1996 to the present. It highlights academic credentials, a three-decade teaching timeline, and a unique teaching philosophy rooted in storytelling and humor.

### ✨ Key Features
- **Modern Tech Stack:** Built with [Astro](https://astro.build/) for lightning-fast performance.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop using **Tailwind CSS**.
- **Interactive Timeline:** A visual journey through 30+ years of teaching excellence across various prestigious institutions.
- **Netlify Integration:** A fully functional contact form powered by Netlify Forms.
- **Honorary Recognition:** Special spotlight on the "Chukkallo Chandrudu" title conferred by peers.

---

## 🛠️ Tech Stack

- **Framework:** Astro 4.x
- **Styling:** Tailwind CSS
- **Fonts:** Playfair Display (Serif) & Inter (Sans-serif)
- **Deployment:** Netlify
- **Icons:** Heroicons
